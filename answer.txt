explaination:

sed 's/-a$//' file.txt | paste -sd,
1️⃣ sed 's/-a$//' file.txt
sed → Stream Editor, used to transform text in files or input streams.

s/.../.../ → The substitution command: s/pattern/replacement/

-a$ → The pattern to match:

-a → literal string -a

$ → end of line (so it matches -a only if it’s at the end)

// → empty replacement, i.e., delete the matched pattern

file.txt → the file being processed

Effect:
Removes -a from the end of each line.

Example:

css
Copy code
Hello-a          → Hello
Fina-all-a       → Fina-all
Good-attention-a → Good-attention
Andavare-all-alangaram-a → Andavare-all-alangaram
2️⃣ | (pipe)
Takes the output of the first command and passes it as input to the next command.

3️⃣ paste -sd,
paste → normally used to merge lines of files.

-s → serial; instead of merging columns, it merges all lines into one single line

-d, → delimiter; separates lines with , (comma)
